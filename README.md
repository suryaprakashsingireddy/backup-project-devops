Automated Backup System

This project is a graded task that automates the process of creating compressed backups of specified directories using a simple Bash script.

Script Overview

The script automatically creates a .tar.gz backup file from the specified source directory and stores it in a defined backup directory.
Each backup is timestamped to ensure unique and organized file versions.
